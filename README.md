# 💤 LazyVim

My personal setup of [LazyVim](https://github.com/LazyVim/LazyVim).

# To clone config (linux)

backup if needed
'''sh
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
'''

pull
'''sh
git clone https://github.com/Tepoys/PersonalNeovimConfig ~/.config/nvim
'''
